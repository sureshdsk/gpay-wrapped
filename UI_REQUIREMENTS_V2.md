## Animation Guidelines

### Principles
- Subtle and purposeful
- Fast (150-300ms)
- Consistent easing (ease-out for enter, ease-in for exit)

### Animations to Add

**Page Transitions:**
```css
.page-enter { opacity: 0; transform: translateY(10px); }
.page-enter-active { opacity: 1; transform: translateY(0); transition: all 200ms ease-out; }
```

**Modal Enter:**
```css
.modal-enter { opacity: 0; transform: scale(0.95); }
.modal-enter-active { opacity: 1; transform: scale(1); transition: all 150ms ease-out; }
```

**Card Hover:**
```css
.card { transition: box-shadow 200ms, transform 200ms; }
.card:hover { box-shadow: var(--shadow-lg); transform: translateY(-2px); }
```

**Skeleton Pulse:**
```css
@keyframes pulse { 0%, 100% { opacity: 1; } 50% { opacity: 0.5; } }
.skeleton { animation: pulse 2s ease-in-out infinite; }
```
